Andes Neural Network Library User Manual  version3.0
Overview

Andes Neural Network Library (libnn) User Manual contains information about neural network function specifications in Andes neural network library and describes how to use these functions.

The functions in Andes neural network library can be classified into several categories: activation functions, basic math functions, concatenation functions, convolution functions, fully connected functions, pooling functions, softmax functions and util functions.

To call the library functions, just include header files provided by Andes neural network library. These header files define prototypes of functions. They are named according to functioncategories as listed below:

  • riscv_nn_activation.h
  • riscv_nn_basic.h
  • riscv_nn_concatenation.h
  • riscv_nn_convolution.h
  • riscv_nn_fully_connected.h
  • riscv_nn_pooling.h
  • riscv_nn_softmax.h
  • riscv_nn_util.h

For example, if you would like to use activation functions to activate neural network data, you can include riscv_nn_activation.h in your C files.

Andes neural network library defines several data types to properly keep the data during computation. These data types, defined in riscv_math_types.h and quoted below, are q7_t for signed 8-bit integer, u8_t for unsigned 8-bit integer, q15_t for signed 16-bit integer, u16_t for unsigned 16-bit integer, q31_t for signed 32-bit integer, u32_t for unsigned 32-bit integer, q63_t for signed 64-bit integer, u64_t for unsigned 64-bit integer, float32_t for 32-bit single-precision floating-point and float64_t for 64-bit double-precision floating-point respectively:

typedef int8_t q7_t; /* for signed 8-bit integer */
typedef uint8_t u8_t; /* for unsigned 8-bit integer */
typedef int16_t q15_t; /* for signed 16-bit integer */
typedef uint16_t u16_t; /* for unsigned 16-bit integer */
typedef int32_t q31_t; /* for signed 32-bit integer */
typedef uint32_t u32_t; /* for unsigned 32-bit integer */
typedef int64_t q63_t; /* for signed 64-bit integer */
typedef uint64_t u64_t; /* for unsigned 64-bit integer */
typedef float float32_t; /* for single-precision (32-bit) floating-point */
typedef double float64_t; /* for double-precision (64-bit) floating-point */

Andes neural network library defines riscv_nn_activation_fun type, defined in riscv_nn_activation.h, for selecting the activation function:

typedef enum
{
NN_SIGMOID = 0, /* Use sigmoid activation function */
NN_TANH = 1, /* Use tanh activation function */
} riscv_nn_activation_fun;
