Overview

This document contains information about neural network function specifications in AndesAIRE Neural Network Library (the NN Library, also known as libnn) and describes how to use these functions.

The functions can be classiﬁed into several categories: activation, basic math, concatenation, convolution, fully-connected layer, pooling, softmax, and utility.

To call the library functions, simply include header ﬁles provided by the NN Library. These header files define prototypes of functions and are named according to the function categories, as listed below:

  • riscv_nn_activation.h
  • riscv_nn_basic.h
  • riscv_nn_concatenation.h
  • riscv_nn_convolution.h
  • riscv_nn_fully_connected.h
  • riscv_nn_pooling.h
  • riscv_nn_softmax.h
  • riscv_nn_util.h

For example, to use activation functions to activate the neural network data, you can include riscv_nn_activation.h in your C files.

The NN Library defines several data types in riscv_math_types.h (as quoted below) to properly keep the data during computation.

The NN Library also defines enumeration types riscv_nn_activation_fun for selecting the activation function and riscv_nn_upsample_method for selecting the upsampling algorithm.

The definition of riscv_nn_activation_fun is provided below and can be found in riscv_nn_activation.h.

typedef enum
{
NN_SIGMOID = 0, /* Use sigmoid activation function */
NN_TANH = 1, /* Use tanh activation function */
} riscv_nn_activation_fun;

The definition of riscv_nn_upsample_method is provided below and can be found in riscv_nn_util.h.

typedef enum
{
NN_UPSAMPLE_NEAREST = 0,
} riscv_nn_upsample_method;
