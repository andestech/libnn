Descriptions of Functions
Here is a list of all modules:
 Activation FunctionsThe activation functions are used to filter out some input data. They include sigmoid, tanh and ReLU (Rectified Linear Unit) functions
 Basic FunctionsThe basic functions are used to perform element-wise basic arithmetic operations
 Concatenation FunctionsThe concatenation functions are used to concatenate the tensor along the specified axis
 Convolution FunctionsThe convolution functions transform the input matrix into a column vector with im2col, and then use matrix-matrix multiplication to get the convolution result. There are three types of quantization on the convolution outputs: shift-based quantization (named with “sft”), symmetric quantization (named with “sym”), and asymmetric quantization (named with “asym”). The followings illustrate the algorithm flow of respective quantization types where rectangles represent inputs or outputs, circles represent arithmetical operations, solid lines are for required steps and dashed lines are for optional ones
 Fully Connected FunctionsThe fully connected functions multiply the input vector by a weight matrix and add a bias, if any, to the result. The supported combinations of input vector and weight matrix are (signed 8-bit integer, signed 8-bit integer), (unsigned 8-bit integer, signed 8-bit integer), (signed 16-bit integer, signed 8-bit integer), (signed 16-bit integer, signed 16-bit integer) and (half-precision floating point, half-precision floating point)
 Pooling FunctionsThe pooling functions are used to downsample input data. They include max and average pooling functions
 Softmax FunctionsThe softmax functions are exponential functions with base 2
 Utils FunctionsUtils functions are miscellaneous auxiliary tools
